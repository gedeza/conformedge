### Reusable Template Pattern (Examination WIs)

**Metadata Header Table (9 rows):**
1. Category (Lifting Accessories / Appliance / Man Riding)
2. Equipment Description
3. Specification/Legislation (BS/EN/ISO/API references)
4. Application Test (New / Repaired / Modified / Quadrennial)
5. Proof Loads (formula — e.g., "SWL + 100%", "SWL + 25%")
6. Examination Requirement (typically "6 Monthly")
7. Internal Examination (interval or "Not Applicable")
8. Identification (ID No, Serial No, Manufacturer, SWL, CE Marking)
9. Certification (certificate types required)

**Body — Two sections:**
1. **Examination (Thorough Examination):** Step-by-step visual/physical inspection with numbered items, reporting requirements
2. **Test (New/Repaired/Modified):** Proof load testing, post-test examination, marking/stamping, certification issuance

### Proof Load Standards

| Equipment Type | Proof Load |
|----------------|------------|
| Lifting Accessories | SWL + 100% |
| Lifting Machines | SWL + 25% |
| Trolley Jacks | SWL + 50% |
| MEWPs | SWL + 25% |
| Drilling Equipment | Per API formulas |

### Overhaul Work Instructions
- Different template: TOC → Dismantle/Examination → Re-assembly/Test
- Companion Work Record Cards
- Edition-tracked versioning

### Key Patterns for ConformEdge Templates
- 6-monthly thorough examination cycles (LOLER compliance)
- "Competent person" examination requirement
- Consistent defect escalation: withdrawal from service → Defect Report → inform responsible representative
- NDT integration (MPI, PT procedures cross-referenced)

## Commercialization Plan

### Template Packages for ConformEdge

1. **ISO 9001:2015 QMS Starter Pack** — Rebrand policies, flow diagram, and core forms (SD 001-023) as ConformEdge templates available to all tiers
2. **Equipment Inspection Template Library** — Rebrand work instruction template pattern and WRC structure as ConformEdge industry templates for lifting/construction/mining
3. **Equipment/Asset Management Module** — New module built from SD 026/027/030 patterns with calibration tracking, maintenance scheduling, repair history, and automated reminders

### Key Template Patterns to Replicate

1. **9-row metadata header** for work instructions (reusable across any equipment type)
2. **5-step WRC process tracking** (Dismantle → Examine → Pre-close → Test → Final)
3. **NCR 3-section workflow** with 5-Whys root cause analysis
4. **Risk scoring matrix** (Probability × Severity × Frequency with ranked escalation)
5. **Customer satisfaction survey** with threshold-based escalation actions
